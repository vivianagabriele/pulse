# Pulse - Real-time Trend Tracker

Pulse scans the web to show you what's actually trending right now across Film, Music, TV, News, Policy, and Viral topics.

## Features
- 🎬 Real-time trend tracking across 6 categories
- 🔥 Live "talking" counts
- 💬 Sharp, punchy explanations
- 🎯 Vibe labels for each trend
- ❤️ Like and share trends
- 📱 Fully responsive (works on mobile & desktop)

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Create a `.env` file with your Anthropic API key:
REACT_APP_ANTHROPIC_KEY=your-key-here

text
4. Run the app: `npm start`
